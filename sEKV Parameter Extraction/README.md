# sEKV Parameters Extraction for IHP SG13G2

In this folder you will find the Quarto notebooks for the extzraction of the sEKV parameters for

* nMOS transistor including [long](/sEKV%20Parameter%20Extraction/sEKV_IHP130nm_nmos_long.pdf), [medium](/sEKV%20Parameter%20Extraction/sEKV_IHP130nm_nmos_medium.pdf) and [short-channel](/sEKV%20Parameter%20Extraction/sEKV_IHP130nm_nmos_short.pdf).
* pMOS transistor including [long](/sEKV%20Parameter%20Extraction/sEKV_IHP130nm_pmos_long.pdf), [medium](/sEKV%20Parameter%20Extraction/sEKV_IHP130nm_pmos_medium.pdf) and [short-channel](/sEKV%20Parameter%20Extraction/sEKV_IHP130nm_pmos_short.pdf).
* Extrinsic capacitance parameters for [nMOS](/sEKV%20Parameter%20Extraction/sEKV_IHP130nm_nmos_ext_cap.pdf) and [pMOS](/sEKV%20Parameter%20Extraction/sEKV_IHP130nm_nmos_ext_cap.pdf).
